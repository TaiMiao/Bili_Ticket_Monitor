# 使用教程
本脚本可以使用exe文件直接运行也可以使用源码运行，请根据自身情况选择
   
## 一、直接运行
### 你可以直接下载release中发布的合适的exe文件运行
#### 优点
- 无需安装python
- 无需配置
#### 缺点
- #### 无法覆盖所有漫展/演出
- 无法配置
- 如需监控部分冷门展演请参见``二、源码运行``自行配置
